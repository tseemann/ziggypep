# ziggypep

Identify bacterial signal peptides

## Description

I want to make an free, unencumbered tool to
identify bacterial signal peptides that can
be distributed in conda for easy use my everyone.

## Installation

```
conda install -c bioconda -c conda-forge ziggypep
```

## Quick start

```
% ziggypep proteins.faa
```
